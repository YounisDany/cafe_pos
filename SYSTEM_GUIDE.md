## 1. 🌟 نظرة عامة على النظام

**CafePOS** هو نظام نقاط بيع سحابي متكامل مصمم خصيصاً للمقاهي والمطاعم والمقاهي المتخصصة. يوفر النظام واجهة عربية كاملة مع إمكانية تخصيص المظهر والألوان، وإدارة متعددة الفروع، ونظام تقارير تفصيلي.

### التقنيات المستخدمة:
| التقنية | الوصف |
|---|---|
| **Next.js 16** | إطار العمل الرئيسي |
| **TypeScript** | لغة البرمجة |
| **Tailwind CSS 4** | التصميم والتنسيق |
| **Prisma ORM** | إدارة قاعدة البيانات |
| **SQLite** | قاعدة البيانات |
| **shadcn/ui** | مكتبة المكونات |

## 2. ✨ مميزات النظام

### 🎯 نقطة البيع
- واجهة سهلة وسريعة لإنشاء الطلبات
- شاشة تعمل باللمس (Touch-friendly)
- بحث فوري عن المنتجات
- تصفية حسب التصنيفات
- سلة تسوق تفاعلية مع إمكانية تعديل الكميات
- دعم الخصومات
- حساب الضريبة تلقائياً
- طرق دفع متعددة (نقدي / بطاقة)
- طباعة إيصالات حرارية

### 📊 التقارير
- ملخص المبيعات اليومية والأسبوعية والشهرية
- تقارير المنتجات الأكثر مبيعاً
- أداء الكاشيرين
- مخططات بيانية تفاعلية

### 🏢 إدارة الفروع
- إدارة متعددة الفروع من مكان واحد
- تتبع مبيعات كل فرع بشكل منفصل

### 👥 إدارة المستخدمين
- 3 مستويات صلاحيات: مالك، مدير، كاشير
- تفعيل/تعطيل الحسابات

### 🎨 تخصيص المظهر
- ألوان مخصصة تناسب علامتك التجارية
- سمات جاهزة متعددة
- شعار المتجر
- تخصيص الفاتورة (رأس، تذييل، عرض)

## 5. 🛒 دليل الاستخدام — نقطة البيع (POS)

### إنشاء فاتورة جديدة:
1. من القائمة الجانبية، اضغط **"نقطة البيع"**
2. اختر التصنيف من الأعلى (أو "الكل")
3. اضغط على المنتج لإضافته للسلة
4. لتعديل الكمية: استخدم أزرار **(+)** و **(-)** بجانب كل منتج
5. لإضافة خصم: اكتب المبلغ في خانة "الخصم"
6. اضغط **"دفع"** لإتمام العملية

### طرق الدفع:
- **نقدي:** أدخل المبلغ المستلم من العميل، سيظهر الباقي تلقائياً
- **بطاقة:** اضغط "تأكيد الدفع" مباشرة

### طباعة الفاتورة:
بعد الدفع، تظهر نافذة الإيصال مع خيار الطباعة

### البحث عن منتج:
- اكتب اسم المنتج أو رقم SKU في خانة البحث
- أو اضغط **Ctrl+K** من لوحة المفاتيح

## 6. 📈 لوحة التحكم

تعرض ملخص سريع لأداء المتجر:

| البطاقة | المحتوى |
|---|---|
| مبيعات اليوم | إجمالي المبيعات بالريال |
| عدد الفواتير | عدد الفواتير المصدرة |
| متوسط الطلب | متوسط قيمة كل فاتورة |
| أفضل منتج | المنتج الأكثر مبيعاً |

كما تحتوي على:
- **مخطط المبيعات الأسبوعية** — رسم بياني تفاعلي
- **أحدث الفواتير** — آخر 5 فواتير مع تفاصيلها

---

## 7. 📦 إدارة المنتجات

### إضافة منتج جديد:
1. اذهب إلى **"المنتجات"** من القائمة
2. اضغط **"إضافة"** (الزر الأخضر)
3. املأ البيانات:
   - **الاسم** (مطلوب)
   - **SKU** (رقم تعريفي اختياري)
   - **السعر** (مطلوب)
   - **التكلفة** (اختياري - لحساب الربح)
   - **التصنيف** (اختر من القائمة)
4. اضغط **"حفظ"**

### تعديل أو حذف:
- اضغط على أيقونة ✏️ بجانب المنتج للتعديل
- اضغط على أيقونة 🗑️ للحذف (سيطلب تأكيد)

### التصفية:
- بحث بالاسم أو SKU
- تصفية حسب التصنيف

---

## 8. 📁 إدارة التصنيفات

### إضافة تصنيف:
1. اذهب إلى **"التصنيفات"**
2. اضغط **"إضافة تصنيف"**
3. أدخل:
   - **الاسم** (مطلوب)
   - **الوصف** (اختياري)
   - **الأيقونة** — اختر من الأيقونات الجاهزة
4. اضغط **"حفظ"**

### الأيقونات المتاحة:

## 9. 📄 إدارة الفواتير

### عرض الفواتير:
- اذهب إلى **"الفواتير"**
- استخدم الفلاتر: **الكل / مدفوعة / معلقة / ملغاة / مفتوحة**
- ابحث برقم الفاتورة

### تفاصيل الفاتورة:
اضغط على أيقونة 👁️ لعرض:
- قائمة المنتجات
- الكميات والأسعار
- المجموع الفرعي + الضريبة + الخصم + الإجمالي
- طريقة الدفع
- خيار الطباعة

### حالات الفاتورة:
| الحالة | المعنى | اللون |
|---|---|---|
| مفتوحة | فاتورة قيد الإنشائية | أزرق |
| مدفوعة | تم الدفع بنجاح | أخضر |
| معلقة | فاتورة مؤجلة | أصفر |
| ملغاة | تم إلغاؤها | أحمر |

---

## 10. 🏢 إدارة الفروع *(مالك فقط)*

### إضافة فرع:
1. اذهب إلى **"الفروع"**
2. اضغط **"إضافة فرع"**
3. أدخل: الاسم، العنوان، رقم الهاتف
4. اضغط **"حفظ"**

### ملاحظات:
- كل فرع له منتجات وأسعار مستقلة ( BranchProduct )
- يمكن تعيين مستخدمين لفرع معين
- الفواتير مرتبطة بالفرع

---

## 11. 👥 إدارة المستخدمين *(مالك ومدير)*

### إنشاء مستخدم:
1. اذهب إلى **"المستخدمين"**
2. اضغط **"إضافة مستخدم"**
3. أدخل:
   - **الاسم** (مطلوب)
   - **البريد** (مطلوب)
   - **كلمة المرور** (مطلوب للمستخدم الجديد)
   - **الدور**: مالك / مدير / كاشير
   - **الفرع**: اختر الفرع المرتبط بالمستخدم
4. اضغط **"حفظ"**

### تفعيل/تعطيل:
اضغط على أيقونة ❌ أو ✅ بجانب المستخدم لتعطيل أو تفعيل حسابه

### التقارير المتاحة:
| التقرير | المحتوى |
|---|---|
| إجمالي المبيعات | مجموع المبيعات في الفترة المحددة |
| عدد الفواتير | عدد الفواتير المصدرة |
| متوسط الطلب | متوسط قيمة الفاتورة |
| صافي الربح | إجمالي الأرباح |
| أفضل المنتجات | ترتيب المنتجات حسب المبيعات |

### الفترات المتاحة:
- **اليوم** — مبيعات اليوم الحالي

- **هذا الشهر** — من بداية الشهر

### تصفية الفروع *(مالك فقط)*:
يمكن للمالك عرض تقارير فرع معين أو جميع الفروع

## 14. ⚙️ إعدادات النظام *(مالك فقط)*

### 14.1 معلومات المتجر
| الحقل | الوصف |
|---|---|
| اسم المتجر | الاسم المعروض في النظام والفاتورة |
| شعار المتجر | صورة PNG, JPG, SVG — حد أقصى 2MB |
| رقم الهاتف | يظهر في الفاتورة |
| البريد الإلكتروني | البريد الرسمي للمتجر |
| العنوان | عنوان المتجر |
| الرقم الضريبي | رقم التسجيل الضريبي (VAT) |

### 14.2 المظهر والألوان 🎨
تخصيص ألوان النظام بالكامل:

#### سمات جاهزة:
| السمة | الوصف |
|---|---|
| عنبري | اللون الافتراضي — دافئ واحترافي |
| أخضر زمردي | مناسب للصيدليات والعصائر |
| أزرق سماوي | أنيق ورسمي |
| بنفسجي | فخم و عصري |
| وردي | مناسب للمخابز والحلويات |
| أحمر ناري | جذاب و مميز |

#### ألوان مخصصة:
- **اللون الأساسي** — اللون الرئيسي للنظام (أزرار، عناوين، أيقونات)
- **اللون الثانوي** — للنصوص والعناوين الفرعية
- **لون التمييز** — للعناصر المميزة

> 💡 عند تغيير الألوان، يتم تطبيقها **فوراً على كل صفحات النظام** — شريط القائمة، الأزرار، البطاقات، النصوص، التدرجات، والظلال

### 14.3 تخصيص الفاتورة 🧾

| الإعداد | الخيارات |
|---|---|
| عرض الورق | 80 مم (قياسي) أو 58 مم (مصغر) |
| عرض الشعار | تفعيل/تعطيل |
| عرض الضريبة | تفعيل/تعطيل |
| عرض الخصم | تفعيل/تعطيل |

## 16. 🆘 الدعم الفني

### مشاكل شائعة:

| المشكلة | الحل |
|---|---|
| لا أستطيع تسجيل الدخول | تأكد من صحة البريد وكلمة المرور |
| المنتجات لا تظهر | تحقق من تفعيل المنتج والتصنيف |
| الفاتورة لا تطبع | تأكد من إعداد الطابعة في المتصفح |
| الألوان لا تتغير | تأكد من الضغط على "تطبيق الألوان" وحفظ |
| الصفحة لا تحمل | تحقق من اتصال الإنترنت وأعد تحميل الصفحة |
